模拟方式

🚀 GPU 粒子模拟

使用 Compute Shader 在 GPU 上进行粒子物理模拟，支持数万粒子实时计算，包括位置、速度、加速度、生命周期等属性。

💫 碰撞检测

支持粒子与场景几何体的碰撞检测，包括地面碰撞、墙壁反弹等，可配置碰撞响应和能量损失。

🌊 子发射器

粒子可以生成子粒子，形成复杂的级联效果，如爆炸产生火花，火花再产生烟雾等多层效果。

🎨 自定义着色器

支持自定义粒子着色器，实现独特的视觉效果，如颜色渐变、大小变化、纹理动画等。

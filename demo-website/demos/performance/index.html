📦 实例化渲染

使用 GPU 实例化技术，单次绘制调用渲染多个相同几何体，大幅减少 CPU-GPU 通信开销，适合大量重复物体的场景。

🎯 视锥剔除

在渲染前判断物体是否在摄像机视野内，剔除不可见物体，减少不必要的绘制调用，提升渲染效率。

👁 遮挡剔除

使用硬件遮挡查询或软件算法，检测被其他物体完全遮挡的物体，避免渲染不可见的几何体。

📐 LOD 系统

根据物体与摄像机的距离，动态切换不同细节级别的模型，远距离使用低模，近距离使用高模，平衡质量与性能。

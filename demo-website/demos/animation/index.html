🦴 GPU 蒙皮

使用 GPU 进行顶点蒙皮计算，支持大量骨骼同时影响顶点，大幅提升动画性能，支持每顶点最多 4 根骨骼权重。

🌳 动画混合树

基于状态机的动画混合系统，支持 1D/2D 混合空间，实现平滑的动作过渡，如行走 - 奔跑的无缝切换。

😊 表情混合形状

支持 Morph Target 表情系统，可混合多种面部表情，实现丰富的角色表情动画，支持权重曲线控制。

🔄 逆向运动学

CCD 和 FABRIK 算法实现实时 IK，用于脚部贴合地面、手部抓取等场景，支持多关节链式求解。

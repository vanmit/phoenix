## 认证概述

Common Criteria (CC) 是国际公认的信息技术安全评估标准 (ISO/IEC 15408)。

EAL4+ 级别要求:
- 系统设计、开发和测试的完整文档
- 独立的安全测试
- 脆弱性分析
- 配置管理
- 生命周期支持

## 🔄 迭代管理流程

### 每个迭代周期

1. **任务分配** → 创建开发 Agent 执行修复
2. **开发修复** → Agent 实施代码修改
3. **质量审核** → 独立审核 Agent 验证
4. **审核结论**:
   - ✅ 通过 → 进入下一个迭代
   - ⚠️ 有条件通过 → 修复建议项后进入下一步
   - ❌ 不通过 → 返回步骤 1 重新修复

### 角色分工

- **管理者** (当前 Agent): 调度、协调、进度跟踪
- **开发 Agent**: 实施代码修复
- **审核 Agent**: 独立质量审核
